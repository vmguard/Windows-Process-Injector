# Windows Process Injector

A Windows-based DLL injector built to explore Windows internals, 
process management, and memory manipulation.

## Features
- Process enumeration
- Manual injection by PID
- Automatic injection on process startup
- Detailed debug logging
- Safe reinjection prevention

## Learning Goals
- Windows API internals
- Virtual memory management
- Process/thread manipulation
- Debug-oriented software design
